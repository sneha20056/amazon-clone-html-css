# Amazon Clone (HTML & CSS)

This is a **clone of the Amazon website** built using **HTML and CSS**. It replicates the layout, design, and style of Amazon's homepage, including:  

- Navigation bar with search and account options  
- Banner for deals and offers  
- Product sections with discounts and images  
- Responsive design elements
